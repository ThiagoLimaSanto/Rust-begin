# 🦀 Início dos Estudos em Rust

Este repositório marca o começo da minha jornada de aprendizado na linguagem **Rust**.

## 🚀 Objetivo

Aprender os conceitos fundamentais da linguagem, incluindo:

- Sintaxe básica  
- Gerenciamento de memória (*ownership* e *borrowing*)  
- Uso do `cargo`  
- Criação e execução de projetos  

## 🛠️ Ambiente de Desenvolvimento

- Editor: VS Code  
- Sistema: WSL (Ubuntu)  
- Gerenciador de pacotes: `cargo`  
- Compilador: `rustc`  

## 📦 Primeiro Projeto

O projeto foi criado utilizando:

```bash
cargo new primeiro_projeto_rust
cd primeiro_projeto_rust
cargo run
